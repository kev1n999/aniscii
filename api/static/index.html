Aniscii Documentation

The aniscii is a simple api to get ascii arts from gif file on terminal

How to use

You can use these example commands in your terminal

Bash

curl https://aniscii.onrender.com/route

Kasane Teto example

curl https://aniscii.onrender.com/teto

Returns

Available routes

These are available animation routes
