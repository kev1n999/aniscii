Aniscii Documentation

The aniscii is a simple api to get ascii arts from gif file on terminal

How to use

You can use these example commands in your terminal

Bash

curl https://aniscii.vercel.app/route

Kasane Teto example

curl https://aniscii.vercel.app/teto

Returns
